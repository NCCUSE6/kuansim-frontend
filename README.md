# Kuansim

A society issues tracking system

- [How to join](https://g0v.hackpad.com/--1OaXIxVVPSd)

# Import fake data

check out fake data

```
$ make checkout
```

import to pgrest.

```
$ make import
```

# Start the server
```
$ npm i
$ ./script/server.sh
```
